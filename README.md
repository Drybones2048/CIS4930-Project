# CIS4930-Project

## VideoStreamer
Handles video compression and multicast streaming to clients with JCodec and MulticastSocket

## Library Installation Instructions for VideoStreamer
1. In Intellij, at the top click file > Project Structure
2. Navigate to Project Settings > Libraries
3. Click the plus icon on the top left of the window
4. Click "From Maven..."
5. In the search bar type in "JCodec" and click the magnifying class icon to search
6. Scroll down the list and select "org.jcodec:jcodec:0.2.5" and click the OK button
7. Click OK again and then click "Apply" at the bottom right and click OK 
8. Do the same process again but this time add "org.jcodec:jcodec-javase:0.2.5"
